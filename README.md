# House Price Prediction

A Machine Learning web application built with Streamlit and Linear Regression.

## Features
- Predict house prices
- Uses multiple house features:
  - Bedrooms
  - Bathrooms
  - Sqft Living
  - Sqft Lot
  - Floors
  - Condition
  - View
  - House Age

## Tech Stack
- Python
- Pandas
- Numpy
- Scikit-Learn
- Streamlit
